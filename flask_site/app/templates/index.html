Predicting Breast Cancer Survival Rate

About our product

With the use of a data set from NKI, our team worked together on a product that helps estimate the survival rate of a breast-cancer patient. This is done with the help of models such as SVM's and random forest classification. In addition, our product also recommends treatment plans based on those factors.

Our mission is to assist breast cancer patients by offering a clear, accessible, and simple-to-understand website that examines data provided by the user to estimate survivability rates and displays common treatment options. Our product can potentially make it possible for doctors and patients to individually customize screening and preventative programs, decreasing the likelihood of a late diagnosis.

Machine Learning

Our product uses machine learning modules to make predictions about survivability rates using patient data. These modules include Linear Regression, a Supported Vector Machine, and Random Forest Classification.


Linear Regression

Linear Regression is a system to predict and model the relationship between multiple variables. In our data set, we used linear regression to predict the relationship between survival and time recurrence. Using this, the mean average value (MAE) was calculated to find the average value of these data points and analyze the variability of data. Our MAE was about 8.5559.




SVM




Here we have a confusion matrix which is a type of support vector machine. A confusion matrix is a table that is used to define and display the performance of a classification algorithm. For this matrix, the x-values were age and time recurrence, and the target variable, which is the variable we were trying to predict, was survival rate. Overall, this matrix shows that after training the algorithm it was able to have an accuracy of 80% because out of the 55 values given, it was able to predict 44 correctly.

Random Forest

A Random Forest based model achieves maximum performance on the training data set when evaluated using tenfold cross validation for the complete gene expression. This classifier structures the data into numerous trees and obtains a final result i.e., whether a person is at risk of having recurrence of breast cancer or not. This may aid clinicians in identifying high risk patients, making treatment decisions and can potentially improve individual outcomes.
